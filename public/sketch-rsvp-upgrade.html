The proposal
1
Theme-matched backgrounds
Page background is derived from the couple's chosen card theme: Polaroid Classic → warm linen #ede8df, Garden Bloom → deep forest #1a2e18, Sage Linen → dark sage #222b20, Midnight → near-black navy #050714. The current hardcoded #0d0b08 is removed. The side polaroid frames also match the theme.
2
Info above the form
The invitation card structure is flipped: couple photo → name → date/time/venue → "Your RSVP" divider → form. Guests see who, when and where before they have to fill anything in. The "Your RSVP" divider makes the transition to the interactive section explicit. The current design buries these details in the footer below the form.
3
Side memory polaroids
On desktop (>900px), the page shows a 3-column grid: scattered memory polaroids on each side, RSVP card in the centre. The side polaroids are pulled from the couple's memory library, randomly selected (up to 3 per side), scattered with light rotation offsets. On mobile they hide and the card goes full-width — no layout change needed.
